# CV - Renato Campos González

Currículum vitae personal con estética retro Windows 98, construido con HTML + Bootstrap 5.

## Descripción

CV estático con interfaz de escritorio estilo Windows 98 (usando [98.css](https://jdan.github.io/98.css/)), enriquecido con Bootstrap 5 para layout responsivo e iconos de Bootstrap Icons. Toda la información está contenida en un único archivo HTML.

## Tecnologías usadas

- [98.css](https://jdan.github.io/98.css/) — estética retro Windows 98
- [Bootstrap 5.3](https://getbootstrap.com/) — grid, utilidades y espaciado
- [Bootstrap Icons 1.11](https://icons.getbootstrap.com/) — iconografía
- HTML semántico + CSS personalizado (`style.css`)

## Estructura

```
cv-Html_only/
├── index.html      # CV completo
├── style.css       # Estilos personalizados
└── src/
```
